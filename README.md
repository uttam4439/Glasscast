1. Glasscast is a modern iOS weather application built with SwiftUI, designed using iOS-native principles and a glassmorphism UI for a clean, premium experience. The app focuses on smooth onboarding, secure authentication, and personalized weather insights.
2. The project follows a state-driven architecture, ensuring seamless navigation across onboarding, signup, OTP verification, password setup, and the home dashboard. 3. Glasscast supports dynamic personalization, allowing users to manage favorite cities synced to their account using a scalable relational database design. The app emphasizes performance, clean data modeling, and extensibility for future AI-powered features.

Key Features

SwiftUI-based iOS-native UI with glass effect
Email OTP authentication with secure signup flow

Supabase backend (Auth, Database, RLS)
Dynamic favorite cities synced per user
Scalable and production-ready architecture
